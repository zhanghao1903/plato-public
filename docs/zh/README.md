# Plato 中文文档

Plato 是一个 Task-first 智能工作台。它不是把工作隐藏在长对话里，而是把用户的自然语言目标转成可查看、可调整、可确认、可追踪的任务计划。

这组中文文档面向两类读者：

| 读者 | 建议入口 |
|---|---|
| 普通用户 | [快速开始](usage/quickstart.md)、[常见问题](usage/faq.md)、[隐私与安全](security/privacy-and-safety.md) |
| 招聘方 / 技术评审者 | [工程亮点](engineering/highlights.md)、[英文架构概览](../architecture/overview.md)、[英文信任与审计模型](../architecture/trust-and-audit.md) |
