# Plato 0.1.0 发布说明

Plato `0.1.0` 是第一个公开的 macOS Apple Silicon 本地候选版本。

它用于早期评估 Task-first 产品循环，不是面向大众的正式发行版本。

## 包含内容

- macOS Apple Silicon DMG；
- Plato 桌面应用壳；
- bundled Python sidecar runtime candidate；
- Main Page Task-first control surface；
- public release metadata 和 checksums；
- 公开安全的产品、使用、架构和视觉文档。

## 评估重点

建议用这个版本评估：

- Task-first 模型是否容易理解；
- Main Page 是否能让计划和进度可见；
- ASK 与 confirmation 是否容易区分；
- Audit Page 是否提升信任；
- public docs 是否足够解释产品。

## 已知限制

- 未签名；
- 未公证；
- macOS Gatekeeper 可能拦截普通打开方式；
- 仅提供 macOS Apple Silicon 版本；
- 部分公开截图展示的是开发预览或产品方向；
- 本仓库不是源码镜像。

## 推荐第一次测试

1. 在测试 workspace 中打开 Plato。
2. 创建一个 Session。
3. 输入低风险目标，例如规划个人网站。
4. 检查生成的任务计划。
5. 有活动后打开 Audit Page。
6. 通过 GitHub issues 反馈困惑或问题。
