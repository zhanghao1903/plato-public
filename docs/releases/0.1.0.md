# Plato 0.1.0 Release Notes

Plato `0.1.0` is the first public macOS Apple Silicon local release candidate.

It is meant for early evaluation of the task-first product loop, not broad
consumer distribution.

## Included

- macOS Apple Silicon DMG release asset;
- desktop Plato app shell;
- bundled Python sidecar runtime candidate;
- Main Page task-first control surface;
- public release metadata and checksums;
- public-safe product, usage, architecture, and visual documentation.

## Evaluation Focus

Use this release to evaluate:

- whether the task-first model is understandable;
- whether the Main Page makes planning and progress visible;
- whether ASK and confirmation concepts make sense;
- whether Audit Page evidence improves trust;
- whether public docs explain the product clearly enough.

## Known Caveats

- The app is unsigned.
- The app is not notarized.
- macOS Gatekeeper may block normal opening.
- The release is macOS Apple Silicon only.
- Some public screenshots show development previews or product direction.
- This repository is not a source-code mirror.

## Recommended First Test

1. Open Plato in a test workspace.
2. Start a new session.
3. Describe a low-risk goal, such as planning a personal website.
4. Review the generated task plan.
5. Open Audit Page after activity exists.
6. Report confusing product or release behavior through GitHub issues.
