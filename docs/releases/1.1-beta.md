# Plato 1.1-beta Release Notes

Plato `1.1-beta` is a public macOS Apple Silicon local beta release focused on
making Product 1.1 inspection and usage foundations available for evaluation.

It remains an unsigned, non-notarized local release. Use it for early testing,
not production distribution.

## Added In This Release

### Token Usage Analytics

- Adds local token usage recording and aggregation foundations.
- Exposes workspace/session-level usage summaries for understanding AI runtime
  cost and cache behavior.
- Keeps the public beta focused on visibility; billing and hard budget
  enforcement remain outside this release.
- Related docs: [User guide](../usage/user-guide.md#5-inspect-the-result) and
  [Privacy and safety](../security/privacy-and-safety.md#current-distribution-caveats).

### Precision File Tools

- Adds precision file operation foundations for line-range reads, workspace
  search, hash-guarded line-range replacement, idempotent append, and
  changed-line evidence.
- Improves the product path for AI-assisted file work where users need smaller,
  reviewable changes rather than broad opaque edits.
- Supports stronger audit evidence for file operations.
- Related docs: [Trust and audit](../architecture/trust-and-audit.md#evidence-anchors)
  and [Privacy and safety](../security/privacy-and-safety.md#files-and-workspaces).

### Workspace Inspection: Git, Diff, File Viewer

- Adds workspace inspection surfaces for repository status, structured diffs,
  and file viewer paths.
- Uses renderer-safe workspace and path labels so public UI does not expose raw
  local filesystem paths.
- Enables users to move from task results into changed files and evidence
  without leaving the local app context.
- Related docs: [Trust and audit](../architecture/trust-and-audit.md#workspace-inspection-direction)
  and [Workspace inspection screenshot](../../assets/screenshots/plato-workspace-inspection.png).

## Known Caveats

- The app is unsigned.
- The app is not notarized.
- macOS Gatekeeper may block normal opening.
- The release is macOS Apple Silicon only.
- This repository is not a source-code mirror.
- Some Product 1.1 foundations are still early: automation policy, broad budget
  enforcement, and marketplace-style distribution are not included.

## Recommended First Test

1. Open Plato in a test workspace.
2. Create or open a local project with Git initialized.
3. Start a low-risk task and inspect the visible task plan.
4. Open workspace inspection to review git status, a changed file, and a diff.
5. Check Audit Page evidence after activity exists.
6. Review token usage summaries after AI-backed activity.
