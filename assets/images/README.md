# Public Images

This directory contains public-safe Plato diagrams and generated explanatory
images.

| File | Purpose |
|---|---|
| `plato-hero.svg` | README hero and product identity graphic. |
| `plato-three-planes.svg` | Text diagram explaining Inspiration, Control, and Trust plane characteristics and relationships. |
| `plato-product-flow.svg` | Product journey from intent to outcome. |
| `plato-architecture-overview.svg` | Public-safe architecture boundary diagram. |
| `plato-trust-flow.svg` | Trust and audit evidence flow diagram. |
| `plato-og-image.svg` | Social/repository preview image. |
| `plato-context-governance.svg` | Context governance diagram. |
| `plato-control-trust-plane.svg` | Main Page control plane and Audit Page trust plane comparison. |
| `plato-macos-local-release.svg` | macOS local release caveat and install flow diagram. |
| `plato-public-docs-map.svg` | Public repository documentation map. |

Product screenshots live in [`../screenshots`](../screenshots).

Do not add images with private paths, local usernames, real user content, API
keys, raw diagnostics, or private repository URLs.
