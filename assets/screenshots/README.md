# Public Screenshots

This directory contains public-safe Plato product screenshots.

| File | Purpose |
|---|---|
| `plato-main-page.png` | Sanitized Main Page screenshot from local mock runtime. |
| `plato-authoring-ask.png` | Public-safe real screenshot showing planning questions before TaskTree generation. |
| `plato-execution-ask.png` | Public-safe real screenshot showing a task-scoped execution question in the detail panel. |
| `plato-audit-page.png` | Sanitized Audit Page screenshot from local mock runtime. |
| `plato-workspace-inspection.png` | Sanitized Workspace Inspection screenshot from a temporary local git workspace. |

Screenshots must not include private paths, local usernames, real user content,
API keys, raw diagnostics, or private repository URLs.
